I like building systems where the hard part is not only the model call, but the surrounding product judgment: permissions, memory, eval, automation, deployment, and the quiet details that make a tool trustworthy enough to use every day.

- Agent memory, user models, and long-running context
- AI-native product interfaces
- Applied AI systems with clear safety and operating boundaries
